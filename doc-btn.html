Buttons

A graphical control element that provides the user a simple way to trigger an event, like searching for a query at a search engine, or to interact with dialog boxes.

Check out Our Custom Buttons styles with support for multiple sizes,states and easy Customization by users.


Default Button

Scratch includes several predefined button styles, each serving its own semantic purpose, with a few extras thrown in for more control

<button type="button" class="btn-blue">Blue</button>

<button type="button" class="btn-green">Green</button>

<button type="button" class="btn-yellow">Yellow</button>

<button type="button" class="btn-red">Red</button>

<button type="button" class="btn-bright">Bright</button>

<button type="button" class="btn-dark">Dark</button>

You can customize these by adding your own CSS class in your code.

Bread Button

<button type="button" class="btn-blue btn-brd">Blue</button>

<button type="button" class="btn-green btn-brd">Green</button>

<button type="button" class="btn-yellow btn-brd">Yellow</button>

<button type="button" class="btn-red btn-brd">Red</button>

<button type="button" class="btn-bright btn-brd">Bright</button>

<button type="button" class="btn-dark btn-brd">Dark</button>

Display Bread buttons by adding "btn-brd" after default button class .

Example class ="btn-yellow btn-brd"

Outline Button

Button that are composed to display in dark background.

<button type="button" class="btn-blue-out">Blue</button>

<button type="button" class="btn-green-out">Green</button>

<button type="button" class="btn-yellow-out">Yellow</button>

<button type="button" class="btn-red-out">Red</button>

<button type="button" class="btn-bright-out">Bright</button>

<button type="button" class="btn-violet-out">Violet</button>

Inverted Button

Formatted for display in dark background.

<button type="button" class="btn-blue-inv">Blue</button>

<button type="button" class="btn-green-inv">Green</button>

<button type="button" class="btn-yellow-inv">Yellow</button>

<button type="button" class="btn-red-inv">Red</button>

<button type="button" class="btn-bright-inv">Bright</button>

<button type="button" class="btn-violet-inv">Violet</button>

The Outline and Inverted buttons can also be displayed as bread button by just adding the class"btn-brd"

<button type="button" class="btn-blue-out btn-brd">Blue</button>

<button type="button" class="btn-green-out btn-brd">Green</button>

<button type="button" class="btn-yellow-inv btn-brd">Yellow</button>

<button type="button" class="btn-red-inv btn-brd">Red</button>

Sizes

Scratch provides you with additional size control of the buttons !!

Large button

Add class="btn-lrg" to increase the size of the button.

<button type="button" class="btn-blue btn-lrg">Blue</button>

<button type="button" class="btn-green btn-lrg">Green</button>

Small button

Add class="btn-sml" to decrease the size of the button.

<button type="button" class="btn-blue btn-sml">Blue</button>

<button type="button" class="btn-green btn-sml">Green</button>
